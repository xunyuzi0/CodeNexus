# CodeNexus (代码枢纽) 

> 突破传统的单机刷题模式，CodeNexus 是一个主打 **“天梯匹配 + 实时竞速”** 的新一代容器化在线算法竞技平台。

## 📖 项目简介

CodeNexus 致力于为开发者提供一个极具沉浸感和竞技性的代码对战环境。我们摒弃了枯燥的单人算法练习，引入了类似电竞的匹配与排位机制。通过高频的实时状态同步与物理级隔离的 Docker 沙箱，玩家可以在这里与全服开发者进行毫秒级响应的在线代码 “1v1” 决斗。

------

## ✨ 核心功能特性

### 🎮 实时天梯匹配与智能对战

- **排位匹配引擎**：支持全服天梯匹配，系统会根据玩家的 Elo 排位积分，动态扩张阈值，精准撮合实力相近的对手。
- **智能抽题**：开局瞬间触发智能分级抽题算法，系统会根据双方历史的 AC（通过）记录，优先抽取两人均未做过或生疏的题目，绝对保障竞技公平性。
- **私人房间模式**：支持创建带有密码的私人对战房间，可自由指定题目邀请好友进行切磋。

### ⚡️ 沉浸式高频动态战场

- **丝滑的分屏工作台**：对战房间内置 `Splitpanes` 引擎，支持左右横纵三栏（题目描述、代码编辑器、战况终端）的自由拖拽缩放。
- **战况雷达与物理动效**：通过高频 WebSocket 实时多播技术，你可以实时看到对手的敲击手速 (CPM)、代码行数 (LOC) 变化。终端会根据对手状态触发视觉特效（如编码时的蓝光脉冲、运行代码时的黄圈特效）。
- **绝对时间引擎**：采用服务端绝对时间戳对齐，免疫浏览器后台休眠导致的时间漂移，确保对战倒计时分秒不差。

### 🛡️ 高性能与高可用体验

- **云端偏好漫游**：深度定制的 Monaco Editor，你的主题、字号等编辑器偏好设置支持云端同步，零侵入持久化到本地。
- **断线重连与防逃跑机制**：具备强大的状态恢复能力。即使 F5 刷新页面也能秒速重回战场。断线提供 5 秒宽恕期，恶意逃跑将触发天梯严惩机制。
- **极速异步评测**：底层由 Docker 沙箱守护，提交代码后不阻塞页面，通过进度条丝滑展示“编译中”、“运行中”等状态。

### 📊 多维度数据大盘与社交体系

- **极客热力图**：个人主页提供类似 GitHub 的代码活跃度热力图，精准记录每日 AC 轨迹。
- **O(logN) 实时排行榜**：提供全球玩家的天梯战力排行榜，支持毫秒级推算你当前的全球排位。
- **题库与收藏系统**：提供海量算法题库过滤，支持创建自定义收藏夹对题目进行分类管理。

------

## 🛠️ 技术栈一览 (Tech Stack)

- **前端架构**: Vue 3 (Composition API) + TypeScript + Vite + Pinia + Vue Router 4。
- **UI 与交互**: Tailwind CSS + Shadcn-vue + Monaco Editor + `@vueuse/motion`。
- **后端架构**: Spring Boot 3 + Java 17 + MyBatis-Plus + MySQL 8。
- **核心中间件**: Redis 集群 (承载匹配池与异步判题队列) + Spring WebSocket。
- **底层评测引擎**: Docker Java API (利用 Linux Cgroups 进行物理级硬件隔离)。

## 🚀 快速启动指南

### 前置环境要求

1. **Java**: JDK 17 及以上版本
2. **Database**: MySQL 8.0
3. **Cache/MQ**: Redis
4. **Environment**: 宿主机必须安装 Docker 并启动守护进程（需拉取 `eclipse-temurin:17-alpine` 镜像）
