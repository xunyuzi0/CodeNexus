# N 皇后

## title
回溯法逐行放置

## content
## 解题思路

### 算法：回溯

逐行放置皇后，对每一行尝试每一列：
- 用三个集合记录已被占用的列、主对角线（row - col）、副对角线（row + col）
- 若当前位置安全，放置皇后并递归下一行
- 回溯时撤销放置

### 复杂度
- 时间：O(N!)
- 空间：O(N)

## code
```java
import java.util.*;
import java.io.*;

public class Solution {

    static List<List<String>> results = new ArrayList<>();
    static int n;

    static void backtrack(int row, int[] queens, Set<Integer> cols, Set<Integer> diag1, Set<Integer> diag2) {
        if (row == n) {
            List<String> board = new ArrayList<>();
            for (int i = 0; i < n; i++) {
                char[] line = new char[n];
                Arrays.fill(line, '.');
                line[queens[i]] = 'Q';
                board.add(new String(line));
            }
            results.add(board);
            return;
        }
        for (int col = 0; col < n; col++) {
            if (cols.contains(col) || diag1.contains(row - col) || diag2.contains(row + col)) continue;
            queens[row] = col;
            cols.add(col);
            diag1.add(row - col);
            diag2.add(row + col);
            backtrack(row + 1, queens, cols, diag1, diag2);
            cols.remove(col);
            diag1.remove(row - col);
            diag2.remove(row + col);
        }
    }

    public static void main(String[] args) throws Exception {
        BufferedReader br = new BufferedReader(new InputStreamReader(System.in));
        n = Integer.parseInt(br.readLine().trim());

        results.clear();
        backtrack(0, new int[n], new HashSet<>(), new HashSet<>(), new HashSet<>());

        StringBuilder sb = new StringBuilder("[");
        for (int i = 0; i < results.size(); i++) {
            if (i > 0) sb.append(",");
            sb.append("[");
            for (int j = 0; j < results.get(i).size(); j++) {
                if (j > 0) sb.append(",");
                sb.append("\"").append(results.get(i).get(j)).append("\"");
            }
            sb.append("]");
        }
        sb.append("]");
        System.out.println(sb);
    }
}
```
