# 二叉树的最近公共祖先

## title
递归分治

## content
## 解题思路

### 算法：递归分治

对每个节点递归判断：
- 若当前节点为 null 或等于 p 或 q，返回当前节点
- 分别在左右子树中查找 p 和 q
- 若 p 和 q 分别在左右子树中找到，当前节点就是 LCA
- 若只在一侧找到，返回那一侧的结果

### 复杂度
- 时间：O(n)
- 空间：O(n) 递归栈

## code
```java
import java.util.*;
import java.io.*;

public class Solution {

    static class TreeNode {
        int val;
        TreeNode left, right;
        TreeNode(int val) { this.val = val; }
    }

    static TreeNode lca(TreeNode root, int p, int q) {
        if (root == null || root.val == p || root.val == q) return root;
        TreeNode left = lca(root.left, p, q);
        TreeNode right = lca(root.right, p, q);
        if (left != null && right != null) return root;
        return left != null ? left : right;
    }

    public static void main(String[] args) throws Exception {
        BufferedReader br = new BufferedReader(new InputStreamReader(System.in));
        String line = br.readLine().trim();
        int p = Integer.parseInt(br.readLine().trim());
        int q = Integer.parseInt(br.readLine().trim());

        // 解析层序遍历数组
        if (line.equals("[]")) return;
        String s = line.substring(1, line.length() - 1);
        String[] parts = s.split(",");

        TreeNode root = new TreeNode(Integer.parseInt(parts[0].trim()));
        Queue<TreeNode> queue = new LinkedList<>();
        queue.offer(root);
        int i = 1;
        while (!queue.isEmpty() && i < parts.length) {
            TreeNode node = queue.poll();
            if (i < parts.length && !parts[i].trim().equals("null")) {
                node.left = new TreeNode(Integer.parseInt(parts[i].trim()));
                queue.offer(node.left);
            }
            i++;
            if (i < parts.length && !parts[i].trim().equals("null")) {
                node.right = new TreeNode(Integer.parseInt(parts[i].trim()));
                queue.offer(node.right);
            }
            i++;
        }

        TreeNode result = lca(root, p, q);
        System.out.println(result.val);
    }
}
```
