# 接雨水

## title
双指针法

## content
## 解题思路

### 算法：双指针

维护左右指针 `left`、`right` 以及左右两侧的最大高度 `leftMax`、`rightMax`：
- 较矮的一侧可以接水，水量 = `maxHeight - height[i]`
- 移动较矮一侧的指针并更新对应最大高度

### 复杂度
- 时间：O(n)
- 空间：O(1)

## code
```java
import java.util.*;
import java.io.*;

public class Solution {
    public static void main(String[] args) throws Exception {
        BufferedReader br = new BufferedReader(new InputStreamReader(System.in));
        String line = br.readLine().trim();
        String s = line.substring(1, line.length() - 1);
        String[] parts = s.split(",");
        int[] height = new int[parts.length];
        for (int i = 0; i < parts.length; i++) height[i] = Integer.parseInt(parts[i].trim());

        int left = 0, right = height.length - 1;
        int leftMax = 0, rightMax = 0;
        int water = 0;

        while (left < right) {
            if (height[left] < height[right]) {
                if (height[left] >= leftMax) leftMax = height[left];
                else water += leftMax - height[left];
                left++;
            } else {
                if (height[right] >= rightMax) rightMax = height[right];
                else water += rightMax - height[right];
                right--;
            }
        }
        System.out.println(water);
    }
}
```
