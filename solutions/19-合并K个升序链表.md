# 合并 K 个升序链表

## title
最小堆合并

## content
## 解题思路

### 算法：优先队列（最小堆）

将所有链表的头节点放入最小堆，每次弹出最小节点加入结果，若该节点有后继则将后继入堆。

输入为二维数组表示的 K 个链表，输出为合并后的数组。

### 复杂度
- 时间：O(N log K)，N 为总节点数
- 空间：O(K)

## code
```java
import java.util.*;
import java.io.*;

public class Solution {
    public static void main(String[] args) throws Exception {
        BufferedReader br = new BufferedReader(new InputStreamReader(System.in));
        String line = br.readLine().trim();

        // 解析二维数组 [[1,4,5],[1,3,4],[2,6]]
        List<int[]> lists = new ArrayList<>();
        if (!line.equals("[]") && !line.equals("[[]]")) {
            String inner = line.substring(1, line.length() - 1);
            int depth = 0;
            StringBuilder buf = new StringBuilder();
            for (char c : inner.toCharArray()) {
                if (c == '[') {
                    depth++;
                    if (depth == 2) buf = new StringBuilder();
                } else if (c == ']') {
                    depth--;
                    if (depth == 1) {
                        String[] parts = buf.toString().split(",");
                        int[] arr = new int[parts.length];
                        for (int i = 0; i < parts.length; i++) arr[i] = Integer.parseInt(parts[i].trim());
                        lists.add(arr);
                    }
                } else if (depth == 2 && c != ',') {
                    buf.append(c);
                }
            }
        }

        // 最小堆：[值, 链表索引, 元素索引]
        PriorityQueue<int[]> pq = new PriorityQueue<>((a, b) -> a[0] - b[0]);
        for (int i = 0; i < lists.size(); i++) {
            if (lists.get(i).length > 0) {
                pq.offer(new int[]{lists.get(i)[0], i, 0});
            }
        }

        List<Integer> result = new ArrayList<>();
        while (!pq.isEmpty()) {
            int[] top = pq.poll();
            result.add(top[0]);
            int listIdx = top[1], elemIdx = top[2];
            if (elemIdx + 1 < lists.get(listIdx).length) {
                pq.offer(new int[]{lists.get(listIdx)[elemIdx + 1], listIdx, elemIdx + 1});
            }
        }

        StringBuilder sb = new StringBuilder("[");
        for (int i = 0; i < result.size(); i++) {
            if (i > 0) sb.append(",");
            sb.append(result.get(i));
        }
        sb.append("]");
        System.out.println(sb);
    }
}
```
