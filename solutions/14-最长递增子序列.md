# 最长递增子序列

## title
贪心 + 二分查找

## content
## 解题思路

### 算法：贪心 + 二分

维护一个数组 `tails`，其中 `tails[i]` 表示长度为 `i+1` 的递增子序列的最小末尾元素：
- 遍历数组，对每个元素用二分查找在 `tails` 中找到第一个 >= 当前元素的位置
- 若位置等于 `tails` 长度，说明需要扩展；否则替换该位置的值

最终 `tails` 的长度即为答案。

### 复杂度
- 时间：O(n log n)
- 空间：O(n)

## code
```java
import java.util.*;
import java.io.*;

public class Solution {
    public static void main(String[] args) throws Exception {
        BufferedReader br = new BufferedReader(new InputStreamReader(System.in));
        String line = br.readLine().trim();
        String s = line.substring(1, line.length() - 1);
        String[] parts = s.split(",");
        int[] nums = new int[parts.length];
        for (int i = 0; i < parts.length; i++) nums[i] = Integer.parseInt(parts[i].trim());

        List<Integer> tails = new ArrayList<>();
        for (int num : nums) {
            int lo = 0, hi = tails.size();
            while (lo < hi) {
                int mid = (lo + hi) / 2;
                if (tails.get(mid) < num) lo = mid + 1;
                else hi = mid;
            }
            if (lo == tails.size()) tails.add(num);
            else tails.set(lo, num);
        }
        System.out.println(tails.size());
    }
}
```
