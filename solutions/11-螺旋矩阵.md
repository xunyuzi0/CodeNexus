# 螺旋矩阵

## title
模拟边界收缩

## content
## 解题思路

### 算法：模拟

维护四个边界 `top`、`bottom`、`left`、`right`，按顺时针方向（右→下→左→上）遍历，每遍历完一行/列就收缩对应边界，直到所有元素都被访问。

### 复杂度
- 时间：O(m × n)
- 空间：O(1)（不算输出）

## code
```java
import java.util.*;
import java.io.*;

public class Solution {
    public static void main(String[] args) throws Exception {
        BufferedReader br = new BufferedReader(new InputStreamReader(System.in));
        String line = br.readLine().trim();

        // 解析二维数组 [[1,2,3],[4,5,6],[7,8,9]]
        line = line.substring(1, line.length() - 1);
        List<int[]> rows = new ArrayList<>();
        int depth = 0;
        StringBuilder row = new StringBuilder();
        for (char c : line.toCharArray()) {
            if (c == '[') {
                depth++;
                if (depth == 2) row = new StringBuilder();
            } else if (c == ']') {
                depth--;
                if (depth == 1) {
                    String[] parts = row.toString().split(",");
                    int[] arr = new int[parts.length];
                    for (int i = 0; i < parts.length; i++) arr[i] = Integer.parseInt(parts[i].trim());
                    rows.add(arr);
                }
            } else if (depth == 2 && c != ',') {
                row.append(c);
            }
        }

        int[][] matrix = new int[rows.size()][rows.get(0).length];
        for (int i = 0; i < rows.size(); i++) matrix[i] = rows.get(i);

        int m = matrix.length, n = matrix[0].length;
        int top = 0, bottom = m - 1, left = 0, right = n - 1;
        List<Integer> result = new ArrayList<>();

        while (top <= bottom && left <= right) {
            for (int j = left; j <= right; j++) result.add(matrix[top][j]);
            top++;
            for (int i = top; i <= bottom; i++) result.add(matrix[i][right]);
            right--;
            if (top <= bottom) {
                for (int j = right; j >= left; j--) result.add(matrix[bottom][j]);
                bottom--;
            }
            if (left <= right) {
                for (int i = bottom; i >= top; i--) result.add(matrix[i][left]);
                left++;
            }
        }

        StringBuilder sb = new StringBuilder("[");
        for (int i = 0; i < result.size(); i++) {
            if (i > 0) sb.append(",");
            sb.append(result.get(i));
        }
        sb.append("]");
        System.out.println(sb);
    }
}
```
