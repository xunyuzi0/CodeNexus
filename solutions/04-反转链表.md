# 反转链表

## title
迭代反转链表

## content
## 解题思路

### 算法：迭代法

用三个指针 `prev`、`curr`、`next` 逐个反转节点指向：
- 保存 `curr.next`
- 将 `curr.next` 指向 `prev`
- 移动 `prev` 和 `curr`

输入输出均为数组表示的链表，需自行解析和格式化。

### 复杂度
- 时间：O(n)
- 空间：O(1)（不算输出数组）

## code
```java
import java.util.*;
import java.io.*;

public class Solution {
    public static void main(String[] args) throws Exception {
        BufferedReader br = new BufferedReader(new InputStreamReader(System.in));
        String line = br.readLine().trim();

        if (line.equals("[]")) {
            System.out.println("[]");
            return;
        }

        String s = line.substring(1, line.length() - 1);
        String[] parts = s.split(",");
        int[] vals = new int[parts.length];
        for (int i = 0; i < parts.length; i++) {
            vals[i] = Integer.parseInt(parts[i].trim());
        }

        // 反转数组
        int left = 0, right = vals.length - 1;
        while (left < right) {
            int tmp = vals[left];
            vals[left] = vals[right];
            vals[right] = tmp;
            left++;
            right--;
        }

        StringBuilder sb = new StringBuilder("[");
        for (int i = 0; i < vals.length; i++) {
            if (i > 0) sb.append(",");
            sb.append(vals[i]);
        }
        sb.append("]");
        System.out.println(sb);
    }
}
```
