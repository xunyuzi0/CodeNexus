# 数组中的第K个最大元素

## title
快速选择算法

## content
## 解题思路

### 算法：快速选择（Quick Select）

基于快速排序的 partition 思想，每次选一个 pivot 将数组分为两部分：
- 若 pivot 的位置恰好是目标位置，返回
- 否则只递归包含目标位置的那一半

平均时间复杂度 O(n)，最坏 O(n^2)（可通过随机化 pivot 避免）。

### 复杂度
- 时间：O(n) 平均
- 空间：O(1)

## code
```java
import java.util.*;
import java.io.*;

public class Solution {

    static int quickSelect(int[] nums, int left, int right, int k) {
        if (left == right) return nums[left];
        int pivot = nums[left + (int)(Math.random() * (right - left + 1))];
        int lo = left, hi = right;
        int mid = left;
        while (mid <= hi) {
            if (nums[mid] > pivot) {
                int tmp = nums[lo]; nums[lo] = nums[mid]; nums[mid] = tmp;
                lo++; mid++;
            } else if (nums[mid] < pivot) {
                int tmp = nums[mid]; nums[mid] = nums[hi]; nums[hi] = tmp;
                hi--;
            } else {
                mid++;
            }
        }
        if (k <= hi && k >= lo) return nums[k];
        else if (k < lo) return quickSelect(nums, left, lo - 1, k);
        else return quickSelect(nums, hi + 1, right, k);
    }

    public static void main(String[] args) throws Exception {
        BufferedReader br = new BufferedReader(new InputStreamReader(System.in));
        String line1 = br.readLine().trim();
        int k = Integer.parseInt(br.readLine().trim());

        String s = line1.substring(1, line1.length() - 1);
        String[] parts = s.split(",");
        int[] nums = new int[parts.length];
        for (int i = 0; i < parts.length; i++) nums[i] = Integer.parseInt(parts[i].trim());

        System.out.println(quickSelect(nums, 0, nums.length - 1, k - 1));
    }
}
```
